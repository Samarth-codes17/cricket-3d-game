# 🏏 Cricket 3D Game

A browser-based 3D cricket game built using Three.js.

## 🚀 Features
- Batting system with timing
- Ball physics (swing + spin)
- Replay camera with slow motion
- Stadium environment
- Shot direction control

## 🎮 Controls
- Space → Hit
- Left/Right → Direction

## 🛠️ Tech Used
- JavaScript
- Three.js
- HTML/CSS
